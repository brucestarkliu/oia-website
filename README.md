# OIA Landing Page

专为中国外贸企业打造的 AI 智能助手产品官网。

## 部署到 GitHub Pages

1. 将此目录推送到 GitHub 仓库
2. Settings → Pages → Source: `main` branch, `/ (root)` folder
3. 在 Custom domain 填写 `oiaagent.com`
4. DNS 添加 CNAME 记录：`oiaagent.com` → `<your-username>.github.io`
5. 等待 DNS 生效（最长 24 小时，通常几分钟）

## 技术栈

- HTML5 + Tailwind CSS (CDN)
- 纯静态，零依赖
- 响应式设计，移动端适配
